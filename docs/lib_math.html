Math functions Matematikai függvények

div
  • Input(s): Bemenet(ek): R0=number, R1=divisor R0=osztandó, R1=osztó
  • Output(s): Kimenet(ek): R4=quotient, R5=reminder R4=eredmény, R5=maradék

Előjel nélküli egész számok osztása: R2= R0/R1. Az osztó (R1) nem lehet 0. Az osztás maradéka az R3 regiszterbe kerül. Dividing unsigned integers: R2= R0/R1. The divisor (R1) cannot be 0. The remainder of the division is placed in the R3 register.

rand
  • Input(s): Bemenet(ek): - -
  • Output(s): Kimenet(ek): R4=next random number R4=következő véletlen szám

32 bites, előjel nélküli véletlen szám generálása. Generate a 32-bit unsigned random number.

rand_max
  • Input(s): Bemenet(ek): R0=max value R0=maximum
  • Output(s): Kimenet(ek): R4=next random number R4=következő véletlen szám

32 bites, előjel nélküli véletlen szám generálása, amely a 0 és a maximum (paraméter) értékhatárok közé esik. Generate a 32-bit unsigned random number that falls between 0 and the maximum (parameter) value limits.

srand
  • Input(s): Bemenet(ek): R0=seed R0=aktuális véletlen szám
  • Output(s): Kimenet(ek): - -

A véletlen szám generátor aktuális értékének beállítása. A következő rand hívás a véletlen szekvencia következő értékét fogja adni. Sets the current value of the random number generator. The next rand call will return the next value of the random sequence.
