Character functions Karakter kezelő függvények

isdigit
  • Input(s): Bemenet(ek): R0=character R0=karakter
  • Output(s): Kimenet(ek): Flag.C=1 if character is number Flag.C=1 ha a karakter számjegy

Az R0 regiszterben lévő karakterkód vizsgálata, hogy az egy számjegy (0-9) ascii kódja-e. Ha igen, a flag regiszter C bitje 1 lesz. Check the character code in register R0 to see if it is the ASCII code for a digit (0-9) character. If so, bit C of register flag will be set to 1.
